Bop, Bop, Boop!

2 Player to Team Challenge Game

Start New Game

    Bop, Bop, Boop!

  • Click start new game
  • Enter the name of both teams or players
  • When the game loads, start game by clicking 'Team/Player One Flip Card'
  • If card equals "Bop", the next player plays and if card does not equal Bop or Boop then the next player loses.
  • If both players play a "BOP", then they must both pay the penalty.
  • If card equals "Booop", the other player automatically loses and has to double the penalty. Unless when the second card is played, it is a "BOP" the player who threw "BOOP" has to take double the shots.
